Info:

This Pack is free
I created a Data Pack that adds full functionality to all the weapons from Valery Download
For information on how to install Data Packs or other information, join this Discord

Features:

RELOADING:
your weapon will automatically reload if your clip reaches 0 but if you want to force reload you can drop the weapon and it will start reloading
(Dropping a weapon will force it to go back into your inventory immediately)

AMMUNITION:
to obtain ammunition for the weapons use /trigger refill
While having arrows in your inventory

UNIQUE FEATURES:
each weapon has it's own unique damage, fire rate and reload speed

How to (un-)install + Commands:

Basic commands:
To enable the data pack
/function prototype_weapons:on

To disable the data pack
/function prototype_weapons:off

If the weapons don't work for a player, that player can try this command (This will reset all ammo and other stats)
/tag @s remove Weapons

Creative commands:
Acquire all weapons
/function prototype_weapons:loadout/loadout

Fill ammunition to full
/function prototype_weapons:ammo/000.1

Extra feature commands
Adds a ammo display for targeted player (V1)
/tag @s add DisplayAmmo1

Adds a ammo display for targeted player (V2)
/tag @s add DisplayAmmo2

Allows player to reload weapons without holding them (Might lag, not fully implemented)
/tag @s add OffReload
